# Thư mục UX_UI

Mục đích: Tổng hợp các file giao diện (UI) của các trang trong dự án để dễ tra cứu, không di chuyển hay đổi cấu trúc mã nguồn hiện tại.

- Không chứa logic mới; chỉ là tài liệu và liên kết tới các file UI đang sử dụng.
- Nếu cần chuyển giao diện sang một chuẩn chung, có thể đề xuất tại đây trước khi refactor.

## Cấu trúc

- `overview.md`: Bản đồ route → file giao diện chính.
- `jobs.md`: Giao diện liên quan đến trang Jobs (danh sách, chi tiết, đã lưu).
- `company.md`: Giao diện trang công ty.
- `admin.md`: Giao diện các trang quản trị (Jobs, Permission, User...).
- `account.md`: Giao diện tài khoản người dùng.
- `layout.md`: Các thành phần layout dùng chung (Header, Menu...).
 - `design-tokens.md`: Chuẩn UI (màu, spacing, typography, radius, shadow) và cách áp dụng.

## Góp ý/Đóng góp

- Thêm hoặc cập nhật liên kết khi có trang mới.
- Mô tả ngắn gọn mục đích UI, nơi load dữ liệu, và các thành phần chính.
